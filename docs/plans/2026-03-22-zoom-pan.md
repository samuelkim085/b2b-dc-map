# Zoom / Pan Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:executing-plans to implement this plan task-by-task.

**Goal:** Add smooth wheel-zoom and drag-pan to the map using d3-zoom, with DC logos and zip dots staying fixed visual size at all zoom levels.

**Architecture:** Attach `d3.zoom()` to the existing SVG ref; store transform `{x,y,k}` in React state; wrap all SVG map content in one `<g transform="translate(x,y) scale(k)">`. Pass `k` down to `DcMarker` (inverse-scale logo), zip dots (inverse-scale radius), Geography strokes, and `FlowLayer` (inverse-scale stroke widths).

**Tech Stack:** d3-zoom v3 (new direct dep), React state, existing svgRef, react-simple-maps Marker/Geographies.

Inside the component, add state:

```ts
const [zoomK, setZoomK] = useState(1)

```

Wait — use a single object to avoid stale closure issues:

```ts
const [zoomXform, setZoomXform] = useState({ x: 0, y: 0, k: 1 })
```

**Step 2: Attach zoom in a useEffect**

Add this effect after the existing TopoJSON fetch effect:

```ts
useEffect(() => {
  if (!svgRef.current) return
  const svg = select(svgRef.current)
  const zoomBehavior = d3zoom<SVGSVGElement, unknown>()
    .scaleExtent([1, 8])
    .on('zoom', (event) => {
      const { x, y, k } = event.transform
      setZoomXform({ x, y, k })
    })
  svg.call(zoomBehavior)
  return () => {
    svg.on('.zoom', null)
  }
}, [svgRef])
```

**Step 3: Verify the page loads without errors**

Open browser devtools console — no errors expected. Scrolling on the map should now update state (verify with a temporary `console.log` inside the handler if needed, then remove it).

### Task 7: Prevent default scroll on the map container

**Files:**
- Modify: `src/components/ShipmentsMap.tsx` or `src/components/ShipmentsMap.css`

d3-zoom calls `event.preventDefault()` on wheel events but only when the SVG is the event target. If the page scrolls instead of the map zooming, add a `wheel` event listener that prevents propagation.

**Step 1: Add passive:false wheel listener in the zoom useEffect**

Inside the zoom `useEffect`, after `svg.call(zoomBehavior)`:

```ts
const el = svgRef.current
const preventScroll = (e: WheelEvent) => e.preventDefault()
el.addEventListener('wheel', preventScroll, { passive: false })
return () => {
  svg.on('.zoom', null)
  el.removeEventListener('wheel', preventScroll)
}
```

**Step 2: Verify**

Hovering the map and scrolling should zoom the map, not the page.
